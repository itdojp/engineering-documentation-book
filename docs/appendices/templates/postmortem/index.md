# ポストモーテムテンプレ

真因と再発防止を「行動」に落とし、フォローアップまで管理します。

## コピペ用テンプレ

```md
## 免責（責任追及しない）

- 個人の責任追及ではなく、仕組みの改善に焦点を当てる

## 事象の要約

- 何が起きたか
- 影響

## 真因（Root Cause）

- 直接原因
- 背後要因（プロセス/設計/運用）

## 再発防止（行動）

- 対策案（いつまでに/誰が/何を/どう検証するか）

## 検知改善

- より早く検知できたか

## 手順改善

- Runbook/手順書の更新点

## フォローアップ管理

- アクション一覧
- 期限とステータス
```

# Runbook テンプレ

平常時/異常時の運用を統一し、判断を短時間で再現します。

## コピペ用テンプレ

## 平常時

- 日次/週次/月次
- 監視項目と閾値

## 異常時

- アラート受信時の初動（確認順）
- 切り分け（範囲縮小/比較/変更点）

## エスカレーション

- いつ/誰に/何を伝えるか
- 必要情報（影響、暫定対応、次アクション）

## 復旧判断

- 暫定 vs 恒久
- ロールバック判断

## 成功条件（確認観点）の例

- 外形: 主要なユーザー操作が成立する（例: ログイン、保存、決済）
- メトリクス: エラー率が基準値に戻る、キュー滞留が解消する
- ログ: 例外の新規発生が止まり、再発兆候がない

## 連絡先

- On-call
- 主管部署

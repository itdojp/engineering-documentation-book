# Runbook テンプレ

平常時/異常時の運用を統一し、判断を短時間で再現します。

## コピペ用テンプレ

## 平常時

- 日次/週次/月次
- 監視項目と閾値

## 異常時

- アラート受信時の初動（確認順）
- 切り分け（範囲縮小/比較/変更点）

## エスカレーション

- いつ/誰に/何を伝えるか
- 必要情報（影響、暫定対応、次アクション）

## 復旧判断

- 暫定 vs 恒久
- ロールバック判断

## 連絡先

- On-call
- 主管部署
